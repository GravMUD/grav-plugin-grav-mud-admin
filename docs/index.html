Install

Requires grav-mud-alpha first. Both plugins are in GPM — free MIT.

bin/gpm install grav-mud-alpha
bin/gpm install grav-mud-admin
bin/grav cache

# user/config/plugins/grav-mud-admin.yaml
enabled: true
access_token: 'your-secret-token'

Open /mud-admin or Admin2 → MUD Editor on Grav 2.0. CodeMirror spec editor — not WYSIWYG.
